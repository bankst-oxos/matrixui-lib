# MatrixUI

## Build requirements (Linux target, Linux host)
 - `libgl-dev`
 - `python3`
 - `libxrandr-dev`
 - `libxinerama-dev`
 - `libxcursor-dev`
 - `libxi-dev`

## Build Requirements (Windows target, Linux host)
 - All packages listed above
 - `mingw-w64`

## Runtime requirements (Linux host)
 - A display
